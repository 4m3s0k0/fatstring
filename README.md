# fatstring
Converting string to tokens.

```
TARGET: '5.0j' -> [Token, ...]

Token
name=5
type=digit
index=0
endidx=1
is_symbol=False
is_multiple=True


Token
name=.
type=dot
index=1
endidx=2
is_symbol=True
is_multiple=True


Token
name=0
type=digit
index=2
endidx=3
is_symbol=False
is_multiple=True


Token
name=j
type=alpha
index=3
endidx=4
is_symbol=False
is_multiple=True
```
